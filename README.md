# bootstrap-component-lab
Framework Integration: Successfully implements Bootstrap 5 via CDN, utilizing its global CSS reset and utility classes to manage layout and typography efficiently.

Complex Navigation: Features a responsive navbar component with a functional "hamburger" toggle for mobile viewports, including dropdown menus and integrated search functionality.

Interactive UI Feedback: Implements a variety of spinner-border components in multiple theme colors (danger, warning, info) to simulate loading states in a modern application.

Component-Based Architecture: Demonstrates the use of advanced Bootstrap components like Toasts and Visually Hidden spans, ensuring the interface is both interactive and accessible.
